## 🎯 Sobre o Projeto
Esta aplicação foi desenvolvida como a solução completa para o **Teste 2** do processo seletivo. O desafio consistiu em duas grandes etapas integradas:
1. **Desenvolvimento (Etapa 1):** Criação de uma interface web reativa e funcional para análise de históricos de chat (`.txt`) utilizando a inteligência artificial da Z.AI.
2. **Qualidade e Automação (Etapa 2):** Aplicação de uma mentalidade rigorosa de Engenharia de Qualidade sobre o produto desenvolvido, estruturando cenários de teste, automação End-to-End (E2E) e documentação estratégica.

## 🛠️ Tecnologias e Arquitetura
* **Frontend:** Angular (Standalone Components, API de Signals para estado reativo performante).
* **Interface:** SCSS (CSS Grid/Flexbox), layout responsivo inspirado no WhatsApp, PrimeNG.
* **Resiliência:** Tratamento de erros robusto no cliente (interceptação de Timeout e Rate Limit 429).
* **Automação QA:** Playwright utilizando TypeScript com injeção de massa de dados externa (Data-Driven via `.csv`).
